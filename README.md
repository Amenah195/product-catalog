# product-catalog

This is a multi-page Product Catalog built with React and React Router v7.
The app displays a list of musical instruments and allows users to view product details using dynamic routing.

## Routes
Home page
Products page
Product details page
404 Not Found Page

## Concepts Used
Nested Routes (<Outlet />)
useParams
useNavigate
NavLink with active style
Local product data (6 items)

Develpoed for week 4 React Router Assignment
